This plugin allows you to use PayPal's Payflow Link payment gateway within your Jigoshop powered WordPress ecommerce site.

== Description ==

If you need to use Payflow Link as your payment gateway this is the plugin for you.

This plugin was created by [Oste Design](http://ostedesign.com/ "Oste Design").

== Installation ==

1. Upload the `jigoshop-paypal-payflowlink-gateway` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to `Jigoshop > Settings > Payment Gateways` to configure the Payflow link gateway settings.
